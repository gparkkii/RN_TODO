# RN_TODO
React Native로 심플한 TODO APP 구현해보기

## Features
1. CRUD 기능
2. AsyncStorage에 데이터 저장 후 새로고침 시에도 데이터 유지하도록 구현
3. `Styled-Components`, `Context API` 이용해서 다크모드 기능 구현
4. `RNDateTimePicker` 로 날짜 Select 기능 추가
